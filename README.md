# Interface Gráfica em Python com PySimpleGUI

Este projeto é uma aplicação simples desenvolvida em **Python** utilizando a biblioteca **PySimpleGUI**, com o objetivo de demonstrar a criação de interfaces gráficas básicas para coleta de dados do usuário.

## Funcionalidades
A aplicação permite:

- Inserir **nome** e **idade**
- Selecionar provedores de e-mail aceitos:
  - Gmail
  - Outlook
  - Yahoo
- Informar se aceita **cartão** (Sim ou Não)
- Ajustar um valor por meio de um **slider**
- Enviar os dados e exibi-los na tela

##  Interface

A interface é composta por campos de texto, checkboxes, radio buttons, slider e um botão de envio, organizados de forma simples e funcional.

##  Tecnologias Utilizadas

- Python 3.9.13
- PySimpleGUI
